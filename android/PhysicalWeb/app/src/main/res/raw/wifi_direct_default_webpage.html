Welcome to Wi-Fi Direct!

Wi-Fi Direct is a peer to peer Wi-Fi network, that allows enabled devices to connect to each other without an access point. The Physical Web team is experimenting with Wi-Fi Direct as a way to take the Physical Web offline. Wi-Fi Direct offers a variety of use cases, such as;

  • An interactive offline web page.
  • File sharing.

When you select a Wi-Fi Direct result from the list of nearby beacons, the app joins your device to the Wi-Fi Direct group. Once connected, the app navigates to the IP and port specified in the Wi-Fi Direct result to open up the broadcasted content.
